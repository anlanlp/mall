# 📖 拼淘狗商城（校园电商 Demo）
## 🔥 基于 Spring Boot + MyBatis + MySQL 开发的简易电商平台

---

# ✨ 项目介绍
**拼淘狗商城** 是模仿拼多多/淘宝风格的小型校园电商系统，前后端分离架构，完整实现电商核心业务流程，适合课程设计、期末大作业参考。

---

# 📌 已实现功能
### 🟢 用户模块
- ✅ 用户注册 / 登录（MD5 加密）
- ✅ 登录状态保持

### 🟢 商品模块
- ✅ 商品列表展示
- ✅ 商品详情页
- ✅ 商品图片完整展示

### 🟢 购物车模块
- ✅ 加入购物车
- ✅ 购物车列表
- ✅ 删除购物车商品
- ✅ **阶梯满减优惠**：满99减9、满129减20
- ✅ 优惠原因&金额清晰展示

### 🟢 订单模块
- ✅ 订单创建
- ✅ 自动生成订单号
- ✅ 记录原价、实付价、优惠信息
